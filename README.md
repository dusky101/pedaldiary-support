# Pedal Diary — Support, Marketing & Privacy site

Static website served by GitHub Pages at <https://pedaldiary.iamcoding.uk/>. It
provides the three URLs that App Store Connect requires for the Pedal Diary
iOS app:

- **Support URL** → <https://pedaldiary.iamcoding.uk/>
- **Marketing URL** → <https://pedaldiary.iamcoding.uk/marketing.html>
- **Privacy Policy URL** → <https://pedaldiary.iamcoding.uk/privacy.html>

## Files

| File | Purpose |
| --- | --- |
| `index.html` | Support page with FAQ accordions and contact details |
| `marketing.html` | Marketing page with features, hero panels and structured data |
| `story.html` | Long-form story behind the app |
| `privacy.html` | Full privacy policy |
| `404.html` | Custom not-found page |
| `styles.css` | Design tokens, seven themes, animations and responsive rules |
| `app.js` | Theme picker (with `localStorage` persistence) and scroll reveal |
| `PQTitle.png` | Title logo shown on every page |
| `favicon-16x16.png`, `favicon-32x32.png`, `favicon.ico`, `apple-touch-icon.png` | Site icons in the sizes browsers and the iOS Home Screen expect |
| `CNAME` | GitHub Pages custom-domain config — contains `pedaldiary.iamcoding.uk` |
